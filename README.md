Windows:

Verificar se o Python está instalado:

    python --version

Instalar o pip (se necessário):

    python -m ensurepip --upgrade

Instalar o Selenium:

    pip install selenium

Linux (Debian/Ubuntu):

Instalar o Python:

    sudo apt update
    sudo apt install python3

Instalar o pip:

    sudo apt install python3-pip

Instalar o Selenium:

    pip3 install selenium
